# Codex Personalis - Hande Bengü

> *"In the grim darkness of the far future, there is only... portfolio websites."*

A Warhammer 40K Adeptus Ministorum grimoire portfolio built entirely with vanilla HTML, CSS, and JavaScript.

Live: **[handecodes.github.io/WarhammerPortfolio](https://handecodes.github.io/WarhammerPortfolio)**

---

## Features

- Real page-turning book with PageFlip.js
- Gothic arch portrait frame
- Wax seal social buttons
- Real parchment texture
- Battle hymn (Children of the Omnissiah)
- Live GitHub repo fetching via GitHub API
- Fully responsive

## Stack

- Vanilla HTML / CSS / JavaScript
- [PageFlip.js](https://github.com/Nodlik/StPageFlip)
- GitHub REST API
- YouTube IFrame API
- GitHub Pages
